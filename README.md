# 📊 Powerbi-Awesome-Chocolates-Dashboard

## 🚀 Project Overview

This project is an interactive Power BI dashboard designed to analyze sales, profit, shipment trends, and performance across different regions, products, and sales teams.

The dashboard helps in understanding business performance and making data-driven decisions.

---

## 📌 Key Features

* 📈 Sales and Profit Analysis by Region
* 📦 Shipment Trend Analysis (Year & Month)
* 🧑‍💼 Top N Sales Persons Performance
* 💰 Profit Calculation & Profit Percentage
* 📊 Product-wise Total Amount Analysis
* 🔄 Year-over-Year Comparison

---

## 📊 Dashboard Pages

### 1️⃣ Introduction Page

* Product-wise sales distribution
* Team contribution breakdown
* Interactive slicers for filtering

### 2️⃣ Profit Analysis

* Total profit by country
* Product-level profit insights

### 3️⃣ Shipment Analysis

* Monthly shipment trends
* Team-wise shipment distribution

### 4️⃣ Amount per Box

* Salesperson performance
* Heatmap analysis by region

### 5️⃣ Top Salespersons

* Top and bottom performers
* Profit percentage comparison

### 6️⃣ Year-over-Year Trends

* Monthly comparison with last year
* Growth and decline analysis

### 7️⃣ Product Analysis

* Total amount by product
* Best and worst performing products

---

## 🛠️ Tools & Technologies
